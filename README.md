## Pet Love Organization Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/049a967aff464e20982f139fe1cb657b/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Announcements
5. Stray Animals
6. Team Activities
7. Personal Center
8. Memo
9. Donation
10. Stray Animal Favorites
11. Stray Animal Orders
12. Team Activity Favorites
13. Team Activity Registration
14. Volunteer Application


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. User Management
5. Stray Animal Management
6. Announcement Management
7. Donation Management
8. Memo Management
9. Team Activity Management
10. Volunteer Application Management
11. Announcement Type Management
12. Stray Animal Type Management
13. Team Activity Type Management
14. User Type Management
15. Carousel Management
